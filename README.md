# zcy_check
基于深度学习的【中草药识别系统】设计与实现~Python

## 一、介绍
中草药识别系统。本系统基于TensorFlow搭建卷积神经网络算法（ResNet50算法）通过对10中常见的中草药图片数据集（'丹参', '五味子', '山茱萸', '柴胡', '桔梗', '牡丹皮', '连翘', '金银花', '黄姜', '黄芩'）进行训练，得到一个识别精度较高的H5格式模型文件，然后基于Django开发可视化的Web网页操作界面，实现用户上传一张图片识别其名称。
